# 데이터셋 (Data)

## 소스 오브 트루스

[`tools/idols.json`](../tools/idols.json) 이 원본입니다. 각 항목:

```json
{ "name": "정연", "group": "TWICE", "agency": "JYP", "gender": "F", "dob": "1996-11-01", "cat": "K-idol" }
```

| 필드 | 설명 |
|---|---|
| `name` | 활동명(한글) |
| `group` | 그룹명(솔로·배우는 개인명 가능) |
| `agency` | 한국 아이돌: `HYBE`/`SM`/`YG`/`JYP`/`기타` 중 하나. 그 외 카테고리: 자유 하위 라벨 |
| `gender` | `M` / `F` |
| `dob` | **양력** 생일 `YYYY-MM-DD` |
| `cat` | 카테고리(생략 시 `K-idol`) |

빌드 시 `index.html`의 `IDOLS` 배열은 8필드 `[name, group, 일주, 오행, gender, agency, dob(YYYYMMDD), cat]` 로 생성됩니다. 일주·오행은 `dob`에서 계산되므로 `idols.json`에는 저장하지 않습니다.

## 현황

- 총 **555명 / 91그룹**
- 카테고리: 한국 아이돌 546 · 미국 배우 4 · 중국 배우 3 · 일본 아이돌 2
- 한국 아이돌 소속사: HYBE 73 · SM 66 · JYP 60 · YG 40 · 기타 307
- 성별: 남 303 · 여 252

## 카테고리

`cat`으로 인물 유형을 구분합니다: `K-idol`(한국 아이돌) · `J-idol`(일본 아이돌) · `C-actor`(중국 배우) · `US-actor`(미국 배우) · `Etc`(기타).

- **`Etc`(기타) 버킷:** 위 카테고리에 딱 맞지 않는 실존 인물을 임시로 모으는 곳입니다. `Etc` 안에서 일관된 하위 유형이 **30개 가까이 쌓이면 독립 카테고리로 승격**합니다(예: 태국 배우 30명 → `TH-actor` 신설). 소속사 `기타`를 세분화하는 방식과 동일한 층위 확장입니다.

- **노출 임계값:** 각 카테고리는 데이터가 **30개 이상** 쌓여야 UI에 노출됩니다(`CAT_MIN=30`). 그 전까지는 카테고리 선택·검색에서 숨겨지고, 앱은 한국 아이돌 단일 화면으로 보입니다.
- **가상 인물(캐릭터)은 제외:** 사주는 출생 연도가 필요한데 대부분의 캐릭터는 월·일만 있어 계산할 수 없습니다. 실존 인물만 수록합니다.

## 빌드 & 검증

```
node tools/build.js          # idols.json → index.html의 IDOLS 재생성
node tools/build.js --check  # 검증만, 파일 미변경
```

`build.js`는 다음을 보장합니다.

- **일주 2방식 교차검증**: JDN 공식과 '기준일로부터의 날짜차' 방식이 불일치하면 **빌드 중단**.
- **앵커 재검증**: 알려진 6건의 (생일→일주)를 매번 확인.
- **입력 검증**: 잘못된 소속사/성별/날짜, 범위 밖 연도(1940~2015) 거부.
- **중복 제거**: 동일 `name|group` 스킵.

동명이인(서로 다른 그룹의 같은 활동명, 예: 여러 그룹의 '지수'·'하니')은 허용되며, 앱에서 소속 그룹으로 구분됩니다.

## 데이터 추가

1. `tools/idols.json`에 항목 추가(확실히 출처가 있는 **양력** 생일만).
2. `node tools/build.js` — `errors / computeMismatch / anchorFails` 가 모두 빈 배열이어야 함.
3. 헤드리스 스모크 테스트로 JS 에러 0 확인(방법은 [tools/README.md](../tools/README.md)).
4. 커밋.

## 신뢰성 메모

- **일주 계산**은 `lunar-python`(검증된 만세력) 및 2방식 교차검증으로 신뢰도가 높습니다.
- **생년월일 자체**는 위키 등 공개 출처 기준이라, 신인·해외 인물 일부는 오차 가능성이 있습니다.
- 과거 교차검증 과정에서 실제 오류를 발견·수정한 이력이 있습니다(예: ITZY 멤버명, 일부 생일, EXO 도경수 생일).
