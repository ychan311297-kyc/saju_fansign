# 인물 데이터 가이드

소스 오브 트루스는 다음 파일입니다.

```text
tools/idols.json
```

## 데이터 형식

```json
{
  "name": "강유찬",
  "group": "A.C.E",
  "agency": "기타",
  "gender": "M",
  "dob": "1997-12-31",
  "cat": "K-idol"
}
```

## 필드

| 필드 | 필수 | 설명 | 예시 |
|---|:---:|---|---|
| `name` | O | 화면에 표시할 이름 | `강유찬` |
| `group` | O | 그룹 또는 활동 구분 | `A.C.E` |
| `agency` | O | 소속사 또는 분류 | `기타` |
| `gender` | O | 상대 성별 | `M`, `F` |
| `dob` | O | 양력 생년월일 | `1997-12-31` |
| `cat` | O | 카테고리 | `K-idol` |

## 여러 명 추가

```json
[
  {
    "name": "강유찬",
    "group": "A.C.E",
    "agency": "기타",
    "gender": "M",
    "dob": "1997-12-31",
    "cat": "K-idol"
  },
  {
    "name": "스즈키 노부유키",
    "group": "배우",
    "agency": "기타",
    "gender": "M",
    "dob": "1992-10-14",
    "cat": "J-actor"
  }
]
```

## JSON 문법 주의

- 전체는 `[`와 `]`로 감쌉니다.
- 각 인물은 `{`와 `}`로 감쌉니다.
- 마지막 인물을 제외하고 객체 뒤에 쉼표를 넣습니다.
- 문자열은 큰따옴표를 사용합니다.
- 날짜는 `YYYY-MM-DD` 형식입니다.
- 남성은 `M`, 여성은 `F`입니다.
- 동일한 `name + group` 조합은 중복으로 처리됩니다.
