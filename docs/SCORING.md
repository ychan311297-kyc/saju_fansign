# True Bias 점수 구조

## 기본 종합점수

```text
기본 종합점수
= 최애 십신 × 0.35
+ 일지 케미 × 0.18
+ 천간합 × 0.10
+ 오행 밸런스 × 0.10
+ 신강·신약 보완 × 0.12
+ 시주 궁합 × 0.05
+ 지장간 공명 × 0.05
+ 년지·월지 지속성 × 0.05
```

| 항목 | 비중 |
|---|---:|
| 최애 십신 | 35% |
| 일지 케미 | 18% |
| 천간합 | 10% |
| 오행 밸런스 | 10% |
| 신강·신약 보완 | 12% |
| 시주 궁합 | 5% |
| 지장간 공명 | 5% |
| 년지·월지 지속성 | 5% |

---

## 1. 최애 십신

| 십신 | 기본점수 | 팬심 해석 |
|---|---:|---|
| 상관 | 100 | 재능에 압도돼 계속 보고 싶은 최애 |
| 식신 | 96 | 편안하게 애정을 쏟고 오래 아끼는 최애 |
| 정인 | 92 | 보고 있으면 위로받고 충전되는 최애 |
| 편인 | 86 | 독특하고 설명하기 어려운 매력의 최애 |
| 정재 | 78 | 차분하고 꾸준히 챙기고 싶은 최애 |
| 편재 | 74 | 화려하고 순간적으로 확 끌리는 최애 |
| 비견 | 70 | 친구 같고 닮아서 편안한 최애 |
| 겁재 | 62 | 경쟁심·소유욕이 생기기 쉬운 최애 |
| 정관 | 56 | 존경하지만 거리감이 있는 최애 |
| 편관 | 48 | 카리스마는 강하지만 소모적인 최애 |

### 성별 보조 가중치

여성 선택:

- 정관 ×1.10
- 편관 ×1.05

남성 선택:

- 정재 ×1.10
- 편재 ×1.05

미선택:

- 모든 십신 ×1.00

최애 매칭이므로 연애모드보다 약하게 적용합니다.

## 4. 오행 밸런스

사용자의 부족한 오행을 상대의 년·월·일주가 보완하면 가점합니다.

- 부족한 오행 보완: 가점
- 이미 과한 오행 강화: 감점
- 점수 범위: 20~100

---

사용자의 신강·신약 상태와 상대 십신 계열의 균형 효과를 평가합니다.

---

## 6. 시주 궁합

사용자의 시지와 상대의 일지를 비교합니다.

시간 미입력 시 중립점수 60을 사용합니다.

---

## 7. 지장간 공명

상대 일지의 대표 지장간을 사용자의 일간 기준 십신으로 바꾸어 점수화합니다.

겉으로 드러난 첫인상보다 오래 볼수록 발견되는 매력을 표현하는 항목입니다.

---

## 8. 년지·월지 지속성

사용자와 상대의 년지·월지 관계를 함께 평가합니다.

```text
지속성
= 년지 관계 × 45%
+ 월지 관계 × 55%
```

장기적으로 분위기와 패턴이 잘 맞는지를 보는 보조 지표입니다.

---

# 설렘 점수

```text
설렘
= 최애 십신 × 40%
+ 천간합 × 25%
+ 일지 케미 × 20%
+ 시주 궁합 × 10%
+ 지장간 공명 × 5%
```

추가 가점:

- 상관 +5
- 편재·편관 +4
- 일지 충 +6
- 천간합 +3

최종 설렘 점수는 100점을 넘지 않도록 제한합니다.

---

# 안정 점수

```text
안정
= 신강·신약 보완 × 25%
+ 일지 케미 × 25%
+ 오행 밸런스 × 20%
+ 최애 십신 × 15%
+ 년지·월지 지속성 × 10%
+ 시주 궁합 × 5%
```

추가 가점:

- 식신·정인 +5
- 정재 +3
- 육합 +4
- 삼합 +3

최종 안정 점수는 100점을 넘지 않도록 제한합니다.


# 최애 등급

| 최종점수 | 등급 |
|---|---|
| 80 미만 | 천천히 알아갈 최애 |
| 80~89.99 | 오래 볼수록 좋아지는 최애 |
| 90~99.99 | TRUE FAVORITE |
| 100~106.99 | DESTINED BIAS |
| 107 이상 | ULTIMATE BIAS |

# 계산상 한계

- 절기 경계는 일 단위 근사입니다.
- 상대 출생시각은 반영하지 않습니다.
- 지장간은 대표 정기만 사용합니다.
- 신강·신약은 앱용 간이 모델입니다.
- 용신·대운·세운·형·파 등 모든 명리 요소를 포함하지 않습니다.
- 본 점수는 정통 명리학의 표준 공식이 아니라 팬 이벤트용 해석 모델입니다.
